Single Responsibility Principle (SRP)

What is the Single Responsibility Principle?

The Single Responsibility Principle (SRP) states:

A class or module should have only one reason to change.

This ensures that each component in a system is focused, simple, maintainable, and easy to test.

Benefits of SRP

  • Improved maintainability
  • Better testability
  • Enhanced reusability
  • Reduced risk of unexpected side effects

Python Example

❌ Without SRP (Bad Example)

class Employee:
    def __init__(self, name, hours_worked, rate):
        self.name = name
        self.hours_worked = hours_worked
        self.rate = rate

    def calculate_salary(self):
        return self.hours_worked * self.rate

    def save_to_file(self):
        with open("employee.txt", "w") as f:
            f.write(f"{self.name}, {self.hours_worked}, {self.rate}")

✅ With SRP (Good Example)

1️⃣ Employee class (data only)

class Employee:
    def __init__(self, name, hours_worked, rate):
        self.name = name
        self.hours_worked = hours_worked
        self.rate = rate

2️⃣ SalaryCalculator class

class SalaryCalculator:
    @staticmethod
    def calculate(employee: Employee):
        return employee.hours_worked * employee.rate

3️⃣ EmployeeRepository class

class EmployeeRepository:
    @staticmethod
    def save(employee: Employee):
        with open("employee.txt", "w") as f:
            f.write(f"{employee.name}, {employee.hours_worked}, {employee.rate}")

4️⃣ Usage

emp = Employee("Alice", 40, 20)
salary = SalaryCalculator.calculate(emp)
EmployeeRepository.save(emp)

Java Example

❌ Without SRP (Bad Example)

class Invoice {
    private String customerName;
    private double amount;

    public Invoice(String customerName, double amount) {
        this.customerName = customerName;
        this.amount = amount;
    }

    public double calculateTax() {
        return amount * 0.18;
    }

    public void printInvoice() {
        System.out.println("Invoice for: " + customerName);
        System.out.println("Amount: " + amount);
        System.out.println("Tax: " + calculateTax());
    }

    public void saveToFile() {
        System.out.println("Saving invoice to file...");
    }
}

✅ With SRP (Good Example)

1️⃣ Invoice class (data only)

class Invoice {
    private String customerName;
    private double amount;

    public Invoice(String customerName, double amount) {
        this.customerName = customerName;
        this.amount = amount;
    }

    public String getCustomerName() {
        return customerName;
    }

    public double getAmount() {
        return amount;
    }
}

2️⃣ TaxCalculator class

class TaxCalculator {
    public double calculateTax(Invoice invoice) {
        return invoice.getAmount() * 0.18;
    }
}

3️⃣ InvoicePrinter class

class InvoicePrinter {
    private TaxCalculator taxCalculator;

    public InvoicePrinter(TaxCalculator taxCalculator) {
        this.taxCalculator = taxCalculator;
    }

    public void print(Invoice invoice) {
        System.out.println("Invoice for: " + invoice.getCustomerName());
        System.out.println("Amount: " + invoice.getAmount());
        System.out.println("Tax: " + taxCalculator.calculateTax(invoice));
    }
}

4️⃣ InvoiceRepository class

class InvoiceRepository {
    public void save(Invoice invoice) {
        System.out.println("Saving invoice to file...");
        // File or database saving logic here
    }
}

5️⃣ Usage Example

public class Main {
    public static void main(String[] args) {
        Invoice invoice = new Invoice("Alice", 5000);

        TaxCalculator taxCalculator = new TaxCalculator();
        InvoicePrinter printer = new InvoicePrinter(taxCalculator);
        InvoiceRepository repo = new InvoiceRepository();

        printer.print(invoice);
        repo.save(invoice);
    }
}
